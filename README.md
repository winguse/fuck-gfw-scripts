# Fuck GFW scripts for Linux

First run, init:

```bash
$ sudo ./0-install-shadowsocks-libv-and-init
```


In the following runs:

```bash
$ sudo ./fuckgfw
```

And in a seprated terminal:

```bash
$ sudo ./start-ss {config-name-if-you-have}
```

To unfuck:

```bash
$ sudo ./unfuck-gfw
```
